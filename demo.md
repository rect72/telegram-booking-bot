# Демо-кейс: Telegram-бот для заявок

## Задача

Бизнес получает заявки вручную в Telegram.

Клиенты пишут в личные сообщения:

- какую услугу хотят;
- когда удобно;
- свой телефон;
- дополнительные вопросы.

Из-за этого заявки можно потерять или забыть перенести в таблицу.

## Решение

Telegram-бот собирает заявку по шагам:

1. Показывает приветствие.
2. Предлагает выбрать услугу.
3. Спрашивает имя.
4. Спрашивает телефон.
5. Спрашивает удобную дату/время.
6. Спрашивает комментарий.
7. Показывает заявку для подтверждения.
8. Отправляет заявку администратору.
9. Сохраняет заявку в Google Sheets или Excel.

## Результат

Администратор сразу получает уведомление в Telegram.

Все заявки сохраняются в одной таблице.

Это помогает:

- не терять клиентов;
- быстрее отвечать;
- видеть все заявки в одном месте;
- меньше делать вручную.

## Что можно адаптировать под клиента

- изменить список услуг;
- добавить выбор мастера;
- добавить несколько админов;
- добавить статусы заявок;
- подключить Google Sheets;
- добавить оплату;
- сделать напоминания.
