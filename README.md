# Telegram Booking Bot

Telegram-бот для записи, заявок и анкет.

## Для кого

Подходит для малого бизнеса, которому нужно принимать заявки через Telegram:

- салоны красоты;
- мастера;
- репетиторы;
- сервисные компании;
- онлайн-школы;
- консультанты.
## Какая проблема решается

Часто заявки приходят вручную в Telegram, Direct или WhatsApp.
Из-за этого сообщения можно потерять, забыть ответить или не перенести данные в таблицу.

Этот бот помогает собирать заявки по шагам и сохранять их в одном месте.

## Что делает бот

- показывает услуги;
- собирает имя клиента;
- собирает телефон;
- спрашивает удобную дату/время;
- принимает комментарий;
- показывает клиенту заявку перед отправкой;
- отправляет заявку администратору;
- сохраняет данные в Google Sheets или Excel.

## Пример заявки

Клиент выбирает услугу и вводит данные:

text  
Услуга: Консультация  
Имя: Иван  
Телефон: +79990000000  
Время: завтра 15:00  
Комментарий: Хочу узнать стоимость  

Администратор получает уведомление в Telegram.

Заявка также сохраняется в таблицу.

## Стек

- Python
- aiogram 3
- Google Sheets API или openpyxl
- SQLite при необходимости
- dotenv

## Как запустить

1. Установить зависимости:

bash  
pip install -r requirements.txt  

2. Создать файл \.env\.

3. Заполнить переменные:

env  
BOT_TOKEN=your_bot_token  
ADMIN_ID=your_admin_id  
GOOGLE_SHEET_ID=your_google_sheet_id  

4. Запустить проект:

bash  
python main.py  

## Что можно доработать под клиента

- добавить оплату;
- добавить выбор мастера;
- добавить календарь;
- добавить статусы заявок;
- добавить напоминания;
- добавить несколько админов;
- добавить CRM-интеграцию;
- подключить Google Sheets;
- сделать отдельные сценарии под разные услуги.
