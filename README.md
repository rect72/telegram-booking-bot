# Telegram Booking Bot

   Telegram-бот для записи клиентов на услугу.

   ## Возможности

   - Принимает заявку от клиента
   - Отправляет заявку администратору
   - Сохраняет заявку в Google Sheets или Excel

   ## Технологии

   - Python
   - aiogram 3
   - python-dotenv
   - openpyxl
